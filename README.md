# WhoAreYou_ATmega2560
마이크로프로세서 수업 학기말 프로젝트 임베디드

### 개발환경
* ATmega 2560 Arduino
* C++, Sketch
* Websocket, SoftwareSerial

### 사용부품
* ESP8266 ESP-01 와이파이 모듈
* HC-06 블루투스 센서
* MC-38 마그네틱 센서
